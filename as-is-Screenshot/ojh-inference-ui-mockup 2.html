2. 모델 & 프레임워크

🚀 vLLM 설정

YAML 미리보기
apiVersion: apps/v1 kind: Deployment spec: replicas: 2 template: spec: containers: - name: vllm image: vllm/vllm-openai:latest args: - --model=gpt-oss-120b - --tensor-parallel-size=4 - --max-model-len=8192 - --gpu-memory-utilization=0.9 resources: limits: nvidia.com/gpu: 4

YAML 미리보기
spec: containers: - name: sglang image: lmsysorg/sglang:latest args: - python3 - -m - sglang.launch_server - --model-path=llama-3-8b - --tp=2 - --context-length=8192 - --max-running-requests=256

🔺 Triton Inference Server 설정

💡 ONNX / TensorRT / TorchScript / TensorFlow SavedModel / OpenVINO 모델 서빙용

GPU 당 인스턴스 수. KIND_CPU 도 지원

name: "text-classifier-v1" platform: "onnxruntime_onnx" max_batch_size: 32 input [ { name: "input_ids" data_type: TYPE_INT64 dims: [ 512 ] } ] output [ { name: "logits" data_type: TYPE_FP32 dims: [ 2 ] } ] dynamic_batching { preferred_batch_size: [ 4, 8, 16 ] max_queue_delay_microseconds: 100 } instance_group [ { count: 2 kind: KIND_GPU } ] version_policy { latest { num_versions: 1 } }
